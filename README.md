# IaC: Terraform + AWS Beanstalk
Infraestrutura como código com Terraform para hospedar uma API no AWS Beanstalk com Docker.


### Projeto
---

Subir uma infraestrutura para uma API através do Terraform no AWS Beanstalk.

- Amazon Web Services;
- Linux
- Terraform
- Docker
